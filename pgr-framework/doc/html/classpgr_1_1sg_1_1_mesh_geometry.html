Detailed Description

Container for the mesh data.

Holds the complete geometry of the mesh, grouped to materialGroups with single material each. Holds all vertices from all submeshes packed to buffer objects called m_vertexBufferObject, m_normalBufferObject and m_texCoordBufferObject
